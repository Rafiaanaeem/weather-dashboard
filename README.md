# Weather Dashboard 🌦️

A simple weather application built using HTML, CSS, and JavaScript.  
It uses the OpenWeather API to fetch real-time weather data.

---

## 🚀 How to Run

1. Download or clone the project
2. Open `index.html` in any browser
3. Enter a city name and click Search

---

## 🔑 API Setup

This project uses OpenWeather API.

Steps:
1. Create account on OpenWeather
2. Get API key
3. Paste key inside `script.js`

---

## ⚙️ Features

- Real-time weather data
- Search any city
- Weather icon support
- Error handling (invalid city / API issues)
- Loading indicator

---

## 🛠️ Tech Stack

- HTML
- CSS (Glassmorphism UI)
- JavaScript (Fetch API)
